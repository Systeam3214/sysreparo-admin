Políticas e Suporte Institucional

Diretrizes legais, privacidade e atendimento técnico para o sistema Rstark.

Privacidade e Segurança de Dados

A Rstark tem um compromisso absoluto com a segurança e a privacidade das informações. Este sistema utiliza infraestrutura robusta para garantir que os dados de seus clientes, serviços e fluxo financeiro estejam permanentemente protegidos contra acessos não autorizados.

Nossa infraestrutura e proteção:

  • Criptografia Google Firebase: Todo o tráfego de dados do sistema funciona sob camada de segurança HTTPS/SSL ponta-a-ponta, sendo processados e mantidos seguros nos servidores corporativos da Google.
  • Acesso Autenticado: A interface de operações é selada. Nenhum dado é baixado no navegador de um dispositivo que não possua um login autenticado formal no servidor.
  • Cookies e Dados Locais: Usamos o cache do navegador exclusivamente para manter a sua sessão fluída (permitindo reinícios sem necessidade de logar toda vez) e para garantir que o sistema não encerre uma operação caso a rede falhe brevemente.

Termos de Operação

As diretrizes a seguir regulamentam a conduta que deve ser mantida ao acessar as ferramentas do painel:

  • Uso Exclusivo: As credenciais fornecidas são instransferíveis. Você é responsável por manter sua tela de trabalho bloqueada ao se afastar.
  • Segurança: O sistema mantém protocolos anti-manipulação ativados. Qualquer interceptação do console ou adulteração forçada dos scripts encerra a página, emitindo o bloqueio "Vanguard" automático via rede.
  • Manutenção de Registros: O administrador institucional autoriza o banco de dados a auto-ordenar os registros mais recentes. Registros vitais gerados na emissão de laudos técnicos possuem respaldo temporal irrevogável.

Dúvidas Frequentes (FAQ)

Como o sistema funciona sem internet?

O sistema possui um modo offline integrado. Se a sua conexão cair, você poderá continuar acessando as Ordens de Serviço recentes e Clientes cacheados. Assim que a rede for restabelecida, a bolinha no canto inferior direito voltará a ficar "Online" e os dados serão sincronizados automaticamente.

Como faço para emitir a Ordem de Serviço para o cliente?

Após criar e salvar uma OS, clique no botão "Ações" e depois no ícone roxo com a impressora. Isso abrirá a via de impressão devidamente formatada e estilizada, pronta para salvar em PDF ou imprimir na impressora da loja.

O que acontece se o prazo de garantia acabar?

O sistema calcula os dias úteis configurados na hora da entrega da OS. Caso o prazo se exceda, o status no setor Financeiro vai alterar a cor automaticamente de verde ("Válida") para cinza ("Expirada"), informando que não há mais cobertura para o serviço.

O site travou ou parou de carregar dados novos, o que fazer?

Isso geralmente ocorre quando o sistema está recebendo uma **atualização crítica**. Siga estes passos na ordem:

  1. Atualize a página (F5) pelo menos umas 5 vezes seguidas.
  2. Aguarde de 5 a 10 minutos para o servidor estabilizar a nova versão.
  3. Reinicie o seu navegador ou o dispositivo (celular/tablet).
  4. Como último recurso, utilize o botão "Limpar Cache Local" nos Ajustes. Aviso importante: Ao limpar o cache, qualquer dado que ainda não tenha sido sincronizado com a nuvem (operando totalmente offline) poderá ser perdido permanentemente.
